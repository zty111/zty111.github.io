1、Dirichlet $k$-th root

考虑f的n次方，第一项永远为1，质数项为n倍, 其他项为等比数列和，可知
$$f^p = 1$$
于是f的指数可对p取模，有
$$f^{k*inv(k)} = g^{inv(k)} = f$$
可用快速幂求出。时间复杂度$O(nlogn)$

solution
